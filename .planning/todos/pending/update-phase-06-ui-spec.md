# Update Phase 06 UI-SPEC

Add a time-based toggle (e.g., segmented control) to the Total Compensation Calculator in `06-UI-SPEC.md`.

Modes:
- **Current Year:** Jan 1 to Dec 31 of the current calendar year.
- **Next 1 Year:** Today's date to exactly 12 months from today.

This toggle will filter RSU vestings so that only those falling within the selected time window are included in the Total Compensation sum.
