- [x] [COMP-01] Base salary, cash bonuses, employer-paid benefits breakdown

## v2 Requirements (Deferred)

- [ ] User authentication and cloud sync

## Out of Scope

- [Live Bank Sync (e.g., Plaid)] — Deferred to focus on local-first PWA and core layout foundations.
