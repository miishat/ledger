## Interaction Patterns

### Focal Point
The **annualized total compensation hero number** (Display type, 28px bold) is the primary visual anchor on the Compensation page, centered within the `CompHeroWidget`. The donut chart sits beneath it showing component breakdown.

### View Toggle
- Default: Annualized breakdown (donut chart + component cards)
- Toggle: Monthly cash flow timeline (when each component pays out)
- Toggle mechanism: Segmented control (2 options), same pattern as existing Recharts view toggles

### Data Entry Flow
1. User clicks "Add Compensation Package" (empty state) or "Edit Package" (populated state)
2. `CompensationModal` opens as full-screen overlay (mobile) / centered modal (desktop)
3. Modal has tabbed sections: Base & Cash → Equity → Benefits
4. Each section saves independently to Zustand store
5. Modal close returns to updated widget view

### Vesting Schedule
- Preset buttons for common schedules (D-05): "4-Year with 1-Year Cliff", "3-Year, No Cliff", "Custom Schedule"
- Custom override: inputs for total vest period (months), cliff length (months), vesting frequency dropdown (monthly/quarterly)
- Timeline visualization: horizontal Recharts `BarChart` with cliff marker line + cumulative `Line` overlay

### Compare Mode
- Toggle via "Compare Another Offer" button below primary package
- Opens simplified input form (same fields, no chart detail)
- Delta view shows +/− per component in a vertical list
- Color coding: green for positive delta, red for negative, neutral for equivalent
