## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Widget internal padding |
| lg | 24px | Gap between Bento Grid widgets |
| xl | 32px | Section spacing |
| 2xl | 48px | Major layout breaks |
| 3xl | 64px | Page margins |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | theme-bg | Background of the dashboard container |
| Secondary (30%) | theme-surface | Background of individual widgets |
| Accent (10%) | theme-accent | Active states and trend indicators |
| Destructive | #ef4444 | Destructive actions |

Accent reserved for: interactive icons, positive trend percentages, active state highlights.
